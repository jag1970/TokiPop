# TokiPop
A project website that offers Japanese merchandise, toys, snacks and other collectible items. It demonstrates
my abilities in the design, planning and deployment of a webiste to potential employers.
## Why does it exist?
This project was completed early in my journey while still in school and shows what I was able to accomplish. And this
particular project was selected in order to offer individuals an alternative place to shop for items from Japan. It is for
the enthusiast who wants to showcase or experience thier love for anything japanese.
## What tools did I use?
For this particular project, I decided to go a bit outside my comfort zone by using a few new tools I have not experienced. 
I used bootstrap as my css framework and cursor AI for the editor. I used Claude AI to create the structure I wanted and cursor AI to add features like a contact page and hover hamburger menu. It was important to me to understand how to use AI as a tool for future employers. For deployment, I used vercel mainly because it supports framework based projects and wanted to introduce myself to this feature and potentinal future projects.
